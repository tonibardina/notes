# 14. Loyalty, Pricing, and Reward Economies

## Core idea

Loyalty programs and pricing models are incentive systems. They shape customer behavior.

In SaaS, pricing can encourage or discourage:

- Collaboration
- Usage
- Expansion
- Experimentation
- Integration
- Data volume
- Automation
- Team adoption

## SaaS pricing examples

### Per-seat pricing

Encourages:

- Revenue expansion as teams grow

Can discourage:

- Inviting collaborators
- Broad adoption
- Occasional users

Useful when:

- Value scales with number of active users

Risky when:

- Collaboration is essential for product value

### Usage-based pricing

Encourages:

- Pay-for-value alignment
- Expansion with usage

Can discourage:

- Experimentation
- Heavy usage if cost feels unpredictable

Useful when:

- Usage clearly maps to value

Risky when:

- Users cannot predict bills

### Feature-tier pricing

Encourages:

- Upgrades for advanced capabilities

Can discourage:

- Activation if core value is locked too high

Useful when:

- Features map to customer maturity

Risky when:

- Essential value is paywalled too early

## Questions

1. What behavior does the pricing model reward?
2. What behavior does it punish?
3. Does pricing align with customer value?
4. Does pricing create anxiety?
5. Does packaging help or hurt product-led growth?
