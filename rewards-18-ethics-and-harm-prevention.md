# 18. Ethics and Harm Prevention

## Core idea

Reward systems can manipulate as well as motivate. SaaS products should help users become more capable, not more dependent.

## Ethical risks in SaaS

- Notification pressure
- False urgency
- Shame-based health scores
- Manipulative trials
- Hidden usage costs
- Public leaderboards that embarrass users
- Over-optimization for engagement
- Dark patterns in cancellation
- Rewarding admins at the expense of end users
- Rewarding company goals while pretending to reward user success

## Ethical questions

1. Does this reward help the user achieve their own goal?
2. Is the system transparent?
3. Can users opt out?
4. Could this create anxiety or pressure?
5. Could this reward shallow behavior?
6. Could smaller customers be disadvantaged?
7. Would we be comfortable explaining this publicly?
